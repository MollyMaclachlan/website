pipupdater

pipupdater is a small command-line tool for automatically updating outdated pip packages. The basic functionality uses the output of pip list --outdated to automatically update any out-of-date packages.
Out of all my GitHub projects, this is currently the one with the most active development. I am currently in the process of adding configuration options.

Go to repository

Smooth Logger

Smooth Logger is a simple logging library originally written for my personal use. One day I was so lazy that instead of searching for a library that suited my needs, I overflowed back around to being productive and wrote one myself. I've become rather fond of it.

Go to repository

Example usage

import smooth_logger

Log = smooth_logger.Logger("Example")
Log.new("This is a log message!", "INFO")

Smooth Progress

Smooth Progress is a simple progress bar made primarily for my own personal use.
See the description of 'Smooth Logger', to the left, for the origins of this library. They have the same story.

Go to repository

Example usage

import smooth_progress

bar = smooth_progress.ProgressBar(limit=100)
bar.open()
for i in range(0,100):
    bar.increment()
bar.close()

Archived Projects

There are a number of archived projects on my GitHub. These primarily include programs written for r/TranscribersOfReddit users in order to streamline the transcribing process, aid moderation, and clean up profiles.
These include OSCR, which removed listed comments from a user's Reddit profile after a period of time; TADR, which would automatically reply 'done' to the first 'cannot find transcription' message from the subreddit's moderation bot, and TORCF, a moderation tool to detect and remove cloned posts in the subreddit queue.
